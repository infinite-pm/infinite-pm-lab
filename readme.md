infinite-pm-lab is the repository for the [lab.infinite.pm](https://lab.infinite.pm) website.

lab.infinite.pm is a static site: home to [infinite.pm](https://infinite.pm) experimental results, POCs, and digital sandcastles.

The official [infinite.pm](https://infinite.pm) website lives in the [infinite-pm-web](https://github.com/infinite-pm/infinite-pm-web) repository.
